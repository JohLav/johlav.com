This project is a simplified strategy game inspired by Ancient Rome.

Players build a virtual Roman Empire by managing provinces, gathering resources, training legions, and engaging in tactical battles with other players.

The game explores key aspects of empire-building, including diplomacy, technological advancement, and military conquest. Through an engaging user interface, you create an immersive and strategic experience for players while staying true to the principles and themes of Ancient Rome.
